# Phase 35 Retry Principle Content Matrix

Фаза 35 добавляет короткую подсказку `Вспомнить правило` только после неверной попытки или в финальном разборе босса. Подсказки ниже authored, не генерируются, не перечисляют скрытые правильные варианты, не называют точный порядок и опираются на education kit плюс уже существующие prep / feedback / takeaway / trap concepts.
