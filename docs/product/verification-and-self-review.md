## Stop conditions

Stop the agentic loop and return to a brief/plan if:

- the diff becomes too large to review;
- the agent changes neighboring architecture without explicit scope;
- the agent adds backend/Supabase/Pachca/LLM work during a frontend-only phase;
- the agent adds runtime internet search/fetch or server-side artifact storage during local-first content phases;
- the gameplay UI starts exposing implementation details;
- failed-attempt feedback or retry principles reveal unselected correct answers, exact correct answer sets or the exact correct order;
- the agent cannot explain a substantial behavior change;
- verification is missing or hand-waved;
- a changed map landmark icon only makes sense after explanation, looks like line soup/broken artifact in a close crop, or violates `docs/product/map-landmark-icon-style.md`;
- browser QA is unavailable and the agent starts inventing one-off browser automation or temporary QA pages instead of reporting the client limitation;
- the second fix attempt repeats the same failure without new evidence.
