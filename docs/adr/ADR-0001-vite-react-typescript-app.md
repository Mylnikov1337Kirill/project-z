# ADR-0001: Vite React TypeScript приложение

Дата: 2026-05-27

## Статус

Принято

## Решение

Использовать Vite + React + TypeScript для frontend-приложения.

## Контекст

Project Z — приложение с большим frontend-слоем, которому нужны быстрые визуальные итерации. Приложение должно легко деплоиться на Netlify, а React остаётся поддерживаемым выбором для ожидаемых разработчиков.

## Последствия

- SSR в v1 не используем.
- Контент и конфиги миссий можно бандлить вместе с frontend.
- Позже backend-возможности можно добавить через API или serverless functions.
